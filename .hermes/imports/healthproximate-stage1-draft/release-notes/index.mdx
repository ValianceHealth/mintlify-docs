<Update label="2025-10" description="Secure File Upload API v1.0">
  ### Secure File Upload API — initial release

  - Presigned-URL uploads for secure, direct-to-storage file transfer.
  - Organization-based file isolation.
  - Automatic filename sanitization.
  - Multipart upload support for large files.
  - API-key authentication and comprehensive error handling.

  *Sourced from the HealthProximate Secure File Upload API changelog.*
</Update>
